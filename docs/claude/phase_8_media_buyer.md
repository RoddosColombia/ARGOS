# Phase 8 — Media Buyer + Click-to-WhatsApp Ads + Compliance Officer en Pauta

## Objetivo declarado
Ejecutar pauta real con approval gates humanos. Media Buyer crea campañas Meta y Google (incluyendo CTW ads que aterrizan directo en el WhatsApp Agent). Compliance Officer valida spending caps y políticas antes de que ejecute. Aplicación del loop ARGOS completo: Strategist recomienda → Compliance valida → CEO aprueba → Media Buyer ejecuta → métricas alimentan Strategist.

## Pre-requisitos
- System User ARGOS y Service Account ARGOS operativos dentro de BM/MCC RODDOS · permisos para publicar campañas confirmados (extensiones solicitadas en Phase 0 ya aprobadas · si no llegaron en la ventana se escala al CEO)
- Phase 7 cerrada (Strategist maduro + data enriquecida)

## Builds incluidos

- Build 8.2 — Google Ads API OAuth con Service Account ARGOS dedicado dentro de MCC RODDOS existente
- Build 8.3 — Media Buyer Agent
- Build 8.4 — Compliance Officer full (caps spending, copy review, frequency caps en WhatsApp)

## Métricas objetivo
- Primera campaña lanzada con presupuesto $100K aprobada vía briefing
- Compliance vetos correctos en ejemplos de prueba
- CTW ads generan >20% de nuevas conversaciones WhatsApp
- ROAS de pauta digital +30% vs baseline (Objetivo O2)
