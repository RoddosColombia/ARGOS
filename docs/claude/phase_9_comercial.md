# Phase 9 — Comercialización a Clientes Externos

## Objetivo declarado
Abrir ARGOS como SaaS a clientes no-RODDOS. Onboarding flow para nuevos workspaces, pricing tiers, self-service signup, documentación pública, MCP server público para agentes IA, landing comercial en getargos.ai.

## Pre-requisitos
- Phases 0-8 cerradas con operación estable de RODDOS durante ≥3 meses
- Objetivo O2 cumplido: +25% venta, -20% rotación, ROAS +30%, hit rate >60%
- Dominio getargos.ai comprado

## Builds incluidos
- Build 9.1 — Onboarding workflow multi-tenant
- Build 9.2 — Pricing engine + billing (Stripe)
- Build 9.3 — Self-service signup con app review propio
- Build 9.4 — Documentación pública API (docs.getargos.ai)
- Build 9.5 — MCP server público (exponer ARGOS como herramienta para otros agentes)
- Build 9.6 — Landing comercial getargos.ai
- Build 9.7 — Onboarding del primer cliente externo piloto

## Métricas objetivo
- 3+ workspaces externos activos
- MRR inicial > $2K USD
- NPS inicial de clientes > 50
